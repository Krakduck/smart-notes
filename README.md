# 📓 Smart Note — Умные заметки на Python

Удобное десктопное приложение для создания, хранения и удаления заметок с современным графическим интерфейсом.

---

## 🚀 Особенности проекта

*   **Современный UI:** Интерфейс построен на библиотеке `CustomTkinter` с поддержкой темной темы 🌙.
*   **Динамический список:** Автоматическая прокрутка списка заметок с помощью `CTkScrollableFrame` 📜.
*   **Удобное чтение:** Большие заметки автоматически переносятся по словам благодаря `CTkTextbox` 📝.
*   **Надежное хранение:** Все данные сохраняются локально в базе данных SQLite.

---

## 🏗️ Архитектура кода (ООП)

Проект написан с использованием объектно-ориентированного программирования. Классы активно общаются между собой через передачу ссылок на объекты в памяти:

*   `App` — главное окно, управляющее вкладками приложения.
*   `ButtonAdd` — панель добавления новых заметок.
*   `Notes` — панель просмотра со встроенным механизмом ручного обновления списка (`refresh`).
*   `ButtonNote` — класс динамической кнопки для каждой отдельной заметки.
